# Клиент системы учёта организаций (ЛР №5)

Консольное приложение на C# для работы с API системы учёта организаций. Приложение реализует устойчивый клиент с кэшированием данных и поддержкой экспорта в различные форматы.

## 📋 Описание

Приложение предоставляет интерфейс для:
- Просмотра списка организаций с поддержкой пагинации
- Поиска организаций по ИНН
- Экспорта данных в CSV, Excel (.xlsx) и PDF форматы
- Кэширования данных для улучшения производительности
- Отмены операций через Ctrl+C

## 🛠 Технологии

- **.NET 6+** с использованием `Microsoft.Extensions.Hosting`
- **Dependency Injection** для управления зависимостями
- **Async/Await** для асинхронных операций
- **Кэширование** данных на стороне клиента
- **Экспорт данных**: CSV, Excel (ClosedXML), PDF (QuestPDF)

## 🚀 Запуск приложения

### Предварительные требования

- .NET 6 SDK или выше
- Работающий API сервер (указывается в конфигурации)

### Установка и запуск

```bash
# Восстановление зависимостей
dotnet restore

# Сборка проекта
dotnet build

# Запуск приложения
dotnet run
```

### Конфигурация

В файле `Program.cs` укажите URL вашего API:

```csharp
builder.Services.AddOrganizationClient("http://localhost:5147");
```

## 🔧 Особенности реализации

### Кэширование
Сервис `CachedOrganizationService` хранит данные в памяти, уменьшая количество запросов к API.

### Обработка ошибок
- Перехват и отображение исключений API
- Поддержка внутренних исключений (InnerException)
- Graceful shutdown при отмене операции

### Прогресс-бар
Визуальное отображение прогресса при экспорте данных с цветовой индикацией.
